Приложение ToDo

Приложение для списка дел. В приложение можно составлять три разных списка (на разных страницах). Выделять выполненые дела, удалять ненужные дела. Добавлять массив дел, которые отображаются в списке сразу после загрузки приложения. Также раелизовано сохранение данных между перезагрузками страницы с помощью localStorage.
